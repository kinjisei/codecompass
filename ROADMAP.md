## Как задеплоить на Vercel (действия пользователя)

Проект уже готов к деплою (`npm run build` проходит, `vercel.json` на месте, первый
коммит сделан). Осталось связать его с GitHub и Vercel — это делает пользователь:

1. Создать репозиторий на GitHub (пустой, без README) — например `codecompass`.
2. Привязать и запушить (Claude может помочь с командами):
   `git remote add origin <URL>` → `git branch -M main` → `git push -u origin main`.
3. На vercel.com → Add New Project → Import из этого GitHub-репозитория.
4. Vercel сам определит Vite; переменные окружения не нужны (у CodeCompass нет
   секретов — всё хранится в localStorage браузера). Нажать Deploy.
5. Получить адрес вида `codecompass-xxx.vercel.app`, открыть на Samsung S25 в Chrome,
   меню → «Установить приложение». Иконка компаса появится на рабочем столе.

Дальше каждый `git push` будет автоматически публиковать новую версию (см. тему
«Vercel и деплой изнутри» в самом приложении).
