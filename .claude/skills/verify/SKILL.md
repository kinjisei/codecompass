# Проверка CodeCompass вживую

1. `npm run build` — типы + сборка (~2 с). Это НЕ проверка, только ворота.
2. `npm run dev` в фоне → http://localhost:5173 (Vite, поднимается за секунду).
3. Браузер: Playwright с системным Edge — браузеры качать не нужно:
   `npm i playwright` во временной папке, затем
   `chromium.launch({ channel: 'msedge', headless: true })`, вьюпорт 400×850 (как телефон).
4. Что прогонять после изменений:
   - главная: 4 вкладки внизу, шестерёнка → /settings;
   - /courses → курс → урок → квиз (обязательно и верный, и неверный ответ) →
     возврат к курсу: галочка и счёт «квиз N/M»;
   - /cards?category=<id> — карточки темы, оценка «Хорошо» двигает очередь;
   - /glossary?entry=<id> — открывает и скроллит к термину;
   - /settings: экспорт (ловить событие download), импорт битого JSON → понятная ошибка.
5. Прогресс живёт в localStorage (ключи `codecompass.*`) — новый browser context
   = чистое состояние, ничего сбрасывать не надо.
6. Бэкенда нет, весь контент статичен в `src/content` — сеть мокать не нужно.

Гочча: в квизе тексты вариантов длинные — в `getByRole('button', { name: ... })`
бери точную подстроку из контента, не пересказ.
